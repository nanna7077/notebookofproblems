Problem #: Nearest Permutation Number
Tags:
 Contributed by: Sriram
Problem Statement: You will be given a positive number n. You have to first find the numbers that are of the same length as n and can be made from the individual numbers in n. Then, you have to print the number which is nearest to n. Note- The output must not be n itself!

Constraints:
    Output must an int and should be single line only.

Sample Input:
1234

Sample Output:
1243
